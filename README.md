# contact-management-system-c
Menu-driven Contact Management System built in C supporting add, delete, search and display operations with PIN-based authentication and dynamic memory allocation.

# Contact Management System

This project is a simple Contact Management System written in C.

## Features
- Add contacts
- Delete contacts
- Search contacts
- Display all contacts
- PIN based authentication

## Technologies
- C Programming
- Standard Libraries (stdio, stdlib, string)
